# Twig Preview VS Code Extension

![Logo](https://raw.githubusercontent.com/dlhoumaud/twig-preview/refs/heads/main/logo.png)

Extension minimale pour prévisualiser les fichiers Twig et fournir un onglet de variables JSON.

Installation et tests locaux

1. Installer les dépendances:

```bash
npm install
```

2. Builder:

```bash
npm run build
```

3. Lancer en debug depuis VS Code (Run Extension)

Utilisation

- Ouvrir un fichier `.twig` dans l'éditeur.
- Cliquer sur l'icône dans la barre de titre de l'éditeur (ou exécuter la commande `Open Twig Preview`).
- Dans la prévisualisation, ouvrir l'onglet `Variables` et entrer un JSON pour simuler les données dynamiques.
